# Li-Li Ye - Personal Website

A professional personal website showcasing my research, publications, and academic achievements in quantum computing and machine learning.

## 🌐 Live Website

Visit the live website: [liliyequantum.github.io](https://liliyequantum.github.io)

## 📋 Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Professional Navigation**: Easy-to-use navigation menu with smooth scrolling
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Educational background and career timeline
- **Research Focus**: Overview of main research areas
- **Publications**: Comprehensive list of peer-reviewed publications and preprints
- **Awards & Honors**: Display of academic achievements and recognitions
- **Technical Skills**: Organized showcase of technical competencies
- **Contact Section**: Multiple ways to get in touch
- **Dark Theme Navigation**: Professional color scheme with accessibility in mind
- **Smooth Animations**: Subtle animations and transitions for enhanced UX
- **Back-to-Top Button**: Easy navigation for long pages

## 📝 Customization

### Update Personal Information

Edit `index.html` to update:
- Contact email and phone number
- Links to social profiles (GitHub, ResearchGate, ORCID)
- Bio and descriptions

### Modify Colors

Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #3b82f6;
    --accent-color: #06b6d4;
    /* ... other colors ... */
}
```

### Add Your CV

1. Save your CV as a PDF file
2. Create a `files/` directory in the repository
3. Place your CV file there as `CV_LiLiYe.pdf`
4. The download link in the hero section will work automatically

## 🔧 Development
